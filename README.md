# KOWA TRADING website

A static React and Three.js website built with Vite.

## Requirements

- Node.js 22.13 or newer

## Local development

```bash
npm install
npm run dev
```

## Production

On Windows, double-click `启动网站.bat`. The launcher installs packages when
needed, creates a fresh static build, starts the site on port 14100, and opens
the browser.

The site is available locally at `http://localhost:14100/` and, when the server
firewall permits it, at `http://SERVER-IP:14100/`.
