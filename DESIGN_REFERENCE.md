# KŌWA TRADING — Design Baseline

Primary visual reference: https://www.rabenrifaie.com/

All future design changes should preserve this baseline unless the user explicitly requests a new direction:

- editorial, highly minimal layout on a white or warm-white background
- oversized sans-serif headlines with tight tracking and strong line breaks
- very small, widely spaced navigation and utility labels
- generous empty space and asymmetric desktop composition
- thin rules instead of cards, shadows, gradients, or heavy decoration
- restrained black/grey palette with one muted vermilion accent
- rolling-text hover treatments, horizontal marquees, and subtle scroll-led movement
- short, direct bilingual Japanese/English copy
- mobile layouts retain typography-led hierarchy rather than becoming card-heavy

Do not copy the reference studio's name, logo, copy, project images, source code, or proprietary font files. Recreate the design language with original KŌWA TRADING content and implementation.
